# Kuboniku.com



### ディレクトリ構成

```bash
.
├── assets/  ...svgやcssなどのアセットファイル
├── common/　...定数やユーティリティ、汎用コンポーネント、フックなど
├── domain/　...アプリケーションにおけるモデルや型定義など
├── features/
│   ├── pages/ ...ページ固有で利用する限定的機能やコンポーネント群
│   │   └──[pageName]
│   │
│   └── [featureName]
│
├── infra/　...データの永続化や外部サービスのAPIを呼び出す実装クラス
└── pages/　...ページコンポーネント
```
