# Kuboniku.com



## ディレクトリ構成

```bash
.
├── assets/  ...svgやcssなどのアセットファイル
├── features/
│   ├── pages/ ...ページ固有で利用する限定的機能やコンポーネント群
│   │   └──[pageName]
│   │
│   └── [featureName]
│
├── domain/　...アプリケーションにおけるモデルや型定義など
├── infra/　...外部サービスのAPIを呼び出す実装クラス
├── common/　...定数やユーティリティ、横断的に利用するコンポーネント、フックなど
└── pages/　...ページコンポーネント
```
